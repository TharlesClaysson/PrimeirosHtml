CSS Clear

A propriedade clear é usada para controlar o comportamento de elementosflutuantes.

Os elementos após um elemento flutuante fluirão ao redor. Para evitar isso, use a propriedadeclear.

left: Elemento é empurrado para baixo de elementos com floatleft;

right: Elemento é empurrado para baixo de elementos com floatright;

both: Elemento é empurrado para baixo de elementos com float left ou right;

none: Elemento não é empurrado para baixo de elementos comfloat.
