Floaté uma propriedade CSS que usamos para posicionar algo na tela, seja um elemento em bloco ou em linha.

Com a propriedade float definida, um elemento pode ser empurrado para esquerda ou para a direita, permitindo que os outros elementos possam organizar em torno dele.

AfloatpropriedadeCSSespecifica como um elemento deve flutuar.


Utilizado para posicionar elementos na esquerda oudireita

left -O elemento flutua à esquerda docontêiner•right-O elemento flutua à direita do seucontêiner

none -O elemento não flutua (será exibido exatamente onde ocorre no texto). Este é opadrão
